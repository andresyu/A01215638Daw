Preguntas a Responder:


¿Qué ventajas y desventajas tiene la integración de tus aplicaciones web con servicios web desarrollados por terceros?

Las ventajas pueden ser el ahorrar mucho tiempo en la parte de implementacion además de ser más seguro utilizar una API, al ser mas confiable por ser creada por empresas y personas reconocidad. La desventaja principal es el tener que depender de un tercero ya que no se puede tener el control completo de la API en caso de que exista algun error.
