Cuestionario 1: BD vs DBMS

¿Qué es entonces una base de datos - DB?

Es una colección de información organizada de forma logica, que un programa de ordenador pueda seleccionar rápidamente los fragmentos de datos que necesite que ademas de que esten relacionados entre si.[1]

¿En qué casos es conveniente usar bases de datos?

Cuando exista informacion (datos) que se relacionen entre si de forma logica y que esta informacion pueda ser usada para un fin.

¿Qué es un sistema de gestión de base de datos - DBMS?

Es un sistema de administración de bases de datos. Software que controla la organización, almacenamiento, recuperación, seguridad e integridad de los datos en una base de datos. Acepta solicitudes de la aplicación y ordena al sistema operativo transferir los datos apropiados. [2]

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

1- Crear y organizar la Base de datos.
2- Establecer y mantener las trayectorias de acceso a la base de datos de tal forma que los datos puedan ser accesados rápidamente.
3- Manejar los datos de acuerdo a las peticiones de los usuarios.
4- Registrar el uso de las bases de datos.
5- Interacción con el manejador de archivos.
6- Respaldo y recuperación: Consiste en contar con mecanismos implantados que permitan la recuperación fácilmente de los datos en caso de ocurrir fallas en el sistema de base de datos.
7- Control de concurrencia: Consiste en controlar la interacción entre los usuarios concurrentes para no afectar la inconsistencia de los datos.
8- Seguridad e integridad: Consiste en contar con mecanismos que permitan el control de la consistencia de los datos evitando que estos se vean perjudicados por cambios no autorizados o previstos. [3]

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

Un verano que trabaje con mi papa en el que entrevistaba personas (directores, productores, utileros actores etc.) para una produccion que se iba hacer, se necesitaba llevar el registro de cada uno con sus respectivos datos: edad, nombre, trabajos anteriores, etc., creo que le hubiera facilitado muchisimo tiempo y esfuerzo el llevar dichos registros en una base de datos.
